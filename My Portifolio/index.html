Linguagens

HTML

HTML, ou HyperText Markup Language, é a linguagem de marcação padrão usada para criar páginas web e aplicativos web. Ela estrutura o conteúdo na web e permite a inclusão de textos, imagens, links, vídeos, e outros tipos de mídia.

CSS

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para descrever a apresentação de um documento HTML.

JS

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web, que permite a criação de conteúdo dinâmico e interativo em páginas web. Originalmente desenvolvida para ser executada no lado do cliente (navegador), JavaScript é agora também usada no lado do servidor através de ambientes como Node.js.

PHP

PHP é uma linguagem de script de código aberto amplamente utilizada para desenvolvimento web, especialmente para a criação de páginas dinâmicas e interativas. Originalmente criado em 1994 por Rasmus Lerdorf, o PHP é executado no servidor, o que significa que o código é interpretado no servidor web e o resultado é enviado ao navegador do cliente como HTML.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) de código aberto, amplamente utilizado para armazenar, organizar e gerenciar grandes quantidades de dados. Desenvolvido inicialmente pela empresa sueca MySQL AB e agora mantido pela Oracle Corporation, o MySQL é uma das escolhas mais populares para aplicações web e sistemas empresariais.
